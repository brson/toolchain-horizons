# Toolchain Horizons Presentation

An impress.js presentation showcasing the dependency toolchain compatibility experiments.

## Structure

- `index.html` - Main presentation file with slide structure
- `styles.css` - Custom styling for the presentation
- `README.md` - This file

## Viewing the Presentation

To view the presentation, open `index.html` in a web browser or serve it with a local web server:

```bash
just serve-presentation
```

Then navigate to http://localhost:8000

## Navigation

- Arrow keys or space: Next slide
- Arrow left/up: Previous slide
- Home: First slide
- End: Last slide
- P: Toggle presenter console (if available)

## Adding Content

Edit `index.html` to add content to the placeholder slides. Each slide is a `<div>` with class `step` and various data attributes that control positioning and animation.
